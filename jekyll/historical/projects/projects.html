Security Software Site Sat Mar 3 02:20:07 EET 2001

This is another project I've thought to start a few times. A site listing the most secure known software in various categories. It would contain maybe around 5 different products for each category, each product having good descriptions and discussions of it's pros and cons and if it has had some security issues, links to bugtraq or other sites. Email notifications of new security issues would be nice feature. And of course this site would have to be well known so new people who would actually benefit from it would find it.

This isn't any further than my thoughts above, so anyone interested should contact me, especially web site designers :)
