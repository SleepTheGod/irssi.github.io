Scripts

Usage: Copy to ~/.irssi/scripts/ directory (or ~/.irssi/scripts/autorun if you want them to run automatically at startup), you'll need to have perl module loaded (/LOAD perl, you can have it automatically loaded at startup by adding the command to ~/.irssi/startup file). Start the script with /RUN script_name. With CVS irssi, you may need to do /ALIAS run script load.

NOTE: there's more scripts and probably more up-to-date in scripts.irssi.de.

  • awayproxy.pl - Set yourself away when last client disconnects from irssi-proxy + capabilities for sending hilighted messages to you with SMSes when you're away
  • chops.pl - Simulates BitchX's /chops and /nops commands (/WHO-like list)
  • dictcomplete.pl - Tab complete words from dictionary
  • grep.pl - /GREP , works only with internal commands so not that useful
  • hebrew.pl - Reverse printed hebrew texts
  • hilightwin.pl - Print hilighted messages & private messages to window named "hilight"
  • hitcount.pl - Show hit counter of your web page in statusbar
  • loadavg.pl - System load average in statusbar
  • logcompress.pl - Compress log files when they're rotated
  • noticemove.pl - Prints private notices from people in the channel where they are joined with you
  • typofix.pl - When someone uses s/foo/bar/, this really modifies the text
  • usercount.pl - Shows number of users in channel in statusbar
  • greet.pl - Sends message to channel when people in greetlist joins
  • tlock.pl - Locks the given topic to channel
  • nickcolor.pl - Shows each nick with different color
  • xmms.pl - Show what you're currently playing in XMMS
  • sms.pl - /SMS sending with address book, currently .pl-only
  • dispatch.pl - Dispatch commands irssi doesn't know always to server instead of complaining
  • blowjob.pl - Blowfish encryption
  • kicks.pl - Various new options for /KICK and /KICKBAN
  • splitlong.pl - Split long messages into multiple
  • chanpeak.pl - Shows the max. number of joined people channel has had in it's lifetime
  • users.pl - /USERS command - displays user list for active channel
  • ipkb.pl - Kick and ban user's IP
  • sping.pl - Displays the latency for given server
  • linkchan.pl - Link two channels from different networks.
  • doc.pl - Key/description database for querying/adding/removing information
  • beep_beep.pl - Replace beeps with some executed command
  • foo.pl - rot$x encoding and automatic decoding with a variable $x.
  • aidle.pl - antyidle with random time.
  • chanshare.pl - display people who are in more than one channel with you
  • cwho.pl - Cached /WHO-like list
  • paste.pl - Reformat pasted text ready to paste onto channels.
  • ppl.pl - /WHO-like list with filtering options
  • relm.pl - Redirect last msgs
  • ircgallery.pl - Show information from IRC gallery (finnish only)
  • fserve.pl - FServe - file server using DCC
  • getnick.pl - Gets or keeps a nickname
  • country.pl - Print the country name in /WHOIS replies + /COUNTRY command
  • autorealname.pl - Print realname of everyone who join to channels
  • autowhois.pl - /WHOIS all the users who send you a private message.
  • nickserv.pl - Sends password to NickServ when it asks it
  • shitlist.pl - BitchX-like /SHITLIST for deopping/kicking/banning unwanted people
  • thistory.pl - Displays information of the current and last 10 topics
  • mute.pl - Mutes +v people in a channel. In some channel annoying people are set +v
  • ownage.pl - How many nicks you "own" (ie. bragging how many nicks are in channels you're opped)
  • dns.pl - /DNS command
  • colorswap.pl - Swap between green and white format for public messages. I think this helps readability. (screenshot)
  • friends.pl - Friends list - auto-opping and voicing
  • title.pl - Show server and channel name in xterm title
  • autoaway.pl - Autoaway/back capability
  • noisyquery.pl - Prints "Query started with nick in window x" when query windows are created automatically
  • kills.pl - Display kills with more understandable messages
  • tracknick.pl - Nice script to keep in track of the people who change nicks too often ;)
