Want to mirror?

Note: I'm not adding any more mirrors unless there's a good reason (eg. much faster internet inside some country). Also whenever some existing mirror breaks, I'll just drop it. It's a bit too much job to try to keep everyone's mirror working :)

We use rsync for mirroring now. For mirroring web, use:

rsync -avz --delete main.irssi.org::irssiweb /local/path/to/irssi.org

And make sure the web server has write access to irssi.org/temp directory. Another thing to check is that /files/snapshots/ directory works, it requires "option indexes" and you might have disabled it.

For mirroring CVS, use:

rsync -avz --delete main.irssi.org::irssicvs /home/cvs/irssi
rsync -avz --delete main.irssi.org::xirssicvs /home/cvs/xirssi
rsync -avz --delete main.irssi.org::irssimod /home/cvs/irssi-modules

Kamil Andrusz has written some instructions for setting up a CVS server.
